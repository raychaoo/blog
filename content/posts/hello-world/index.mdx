## 写在前面

欢迎来到我的个人技术博客！这是我使用 **Next.js** 和 **MDX** 搭建的静态博客站点。

## 为什么搭建这个博客

1. **记录学习过程** — 把学到的知识整理成文章，加深理解
2. **分享技术心得** — 与更多开发者交流，共同进步
3. **建立个人品牌** — 通过持续输出建立技术影响力

### 关于技术选型

本站采用了以下技术栈：

- **Next.js 15** — React 全栈框架
- **TypeScript** — 类型安全
- **Tailwind CSS** — 原子化 CSS
- **MDX** — 在 Markdown 中使用 React 组件
